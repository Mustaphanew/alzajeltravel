المادة 3

كيف نجمع بياناتك

3.1 عند التسجيل وتسجيل الدخول

نجمع المعلومات التي تُقدِّمها عند إنشاء حساب وكيل أو تسجيل الدخول: الاسم، اسم الوكالة، البريد الإلكتروني، رقم الهاتف، وبيانات الاعتماد الأمنية.

3.2 عند إجراء الحجوزات

نجمع بيانات الركاب المُدخَلة بواسطتك (الاسم، رقم الجواز، تاريخ الميلاد، الجنسية) لإتمام حجوزات تذاكر الطيران.

3.3 البيانات التقنية للجهاز

يجمع التطبيق تلقائياً: رمز الجهاز (Device Token) لخدمة الإشعارات، نوع الجهاز ونظام التشغيل، وعنوان IP.

3.4 الإشعارات والتنبيهات

نستخدم Firebase Cloud Messaging (FCM) وAwesome Notifications لإرسال إشعارات متعلقة بالحجوزات والعروض. تُخزَّن سجلات الإشعارات محلياً على جهازك عبر قاعدة بيانات SQLite.

3.5 التواصل المباشر

إن تواصلت معنا عبر البريد الإلكتروني أو واتساب أو تيليغرام، نحتفظ بسجل للمراسلات لأغراض الدعم الفني.

المادة 4

أمان وتخزين البيانات

نتخذ إجراءات تقنية وتنظيمية مناسبة لحماية بياناتك من الوصول غير المصرح به أو الفقدان أو التلف، وتشمل:

  • تشفير بيانات الجلسة والرمز المميز (Bearer Token) باستخدام Flutter Secure Storage.
  • الاتصال بالخوادم عبر بروتوكول HTTPS فقط.
  • انتهاء الجلسة تلقائياً عند اكتشاف رمز مصادقة غير صالح (401).
  • تخزين الإعدادات المحلية بصورة مشفرة.

نحتفظ بياناتك طوال مدة علاقتك التجارية معنا، أو وفق ما تقتضيه الأنظمة المعمول بها.

المادة 6

مشاركة بياناتك مع أطراف ثالثة

لا نبيع بياناتك الشخصية لأي جهة. قد نشاركها فقط في الحالات التالية:

  • مزودو خدمة الطيران: نمرر بيانات الراكب إلى شركاء GDS وموفري المحتوى الجوي لإتمام الحجوزات.
  • Firebase / Google: لخدمات الإشعارات (FCM) والتحليلات وفق سياسة خصوصية Google.
  • SkyTaap: واجهة API الخلفية (skytaap.net) المستخدمة لمعالجة بيانات الرحلات.
  • الجهات القانونية: في حالة صدور أمر قضائي أو طلب حكومي رسمي.
  • الخلفاء التجاريين: في حالة الاندماج أو الاستحواذ أو نقل الأصول.
